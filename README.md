# Angular 2 (TS) project using Giphy API #

First run:

**npm install**

Then: 

**npm start**

To execute tests:

**npm test**

To build and create the dist/ folder:

**npm run build**
 
This project use Angular 2 with ng-bootstrap for pagination and modals.
